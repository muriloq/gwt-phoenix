Phoenix Arcade Emulator using HTML5

by muriloq

Press 3 to insert coins, then press 1 or 2 to start game (1 or 2 players).

Use the keyboard arrows to move your ship; press down to activate shields, and space to fire.

If your browser is too slow the emulator will increase frame skip automatically (press A to turn this feature on or off.

You can adjust frame skip manually pressing Page Up or Page Down.

Press S to run at the maximum possible speed - it's a nice Javascript benchmark!

Sound support is preliminary; press M to toggle it.

The first version of this emulator was a Java applet written by me in 1997. It was the first arcade emulator written in Java in the world... As far as I know this one is the first arcade emulator written using GWT, too! :-)
